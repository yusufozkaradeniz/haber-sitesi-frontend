# 📰 Haber Sitesi Projesi (Yazılım Mühendisliği Vize Çalışması)

Bu proje; **Spring Boot** (Backend) ve **React** (Frontend) kullanılarak geliştirilmiş, PostgreSQL veritabanı ile çalışan, çok dilli destek sunan tam kapsamlı bir haber platformudur.

## 🛠️ Teknik Özellikler & Gereksinimler
Proje kapsamında aşağıdaki 10 ana gereksinim başarıyla tamamlanmıştır:

1.  **Şifre Sıfırlama:** Gmail SMTP ve Uygulama Şifresi entegrasyonu.
2.  **Profil Yönetimi:** Kullanıcı bilgilerini düzenleme ve baş editör yetkileri.
3.  **Okuma Geçmişi:** Ziyaret edilen haberlerin anlık takibi ve temizlenmesi.
4.  **Haber Kaydetme:** Beğenilen içeriklerin kişisel listeye eklenmesi.
5.  **Bildirim Sistemi:** Yeni haber eklendiğinde anlık görsel uyarılar.
6.  **Yorum Sistemi:** Haberlere yorum yapma ve admin tarafından yönetilmesi.
7.  **Beğeni Mekanizması:** İçeriklerin etkileşim sayısının takibi.
8.  **Çok Dilli Destek:** 5 farklı dilde (TR, EN, DE, ES, FR) anlık çeviri motoru.
9.  **Reklam Yönetimi:** Dinamik reklam ekleme, güncelleme ve silme.
10. **Admin Paneli:** Full CRUD (Ekleme, Okuma, Güncelleme, Silme) operasyonları.

---

## 🏗️ Kullanılan Teknolojiler
* **Backend:** Java 17, Spring Boot, Spring Data JPA
* **Frontend:** React.js, Axios, Google Translate API
* **Veritabanı:** PostgreSQL (Render)
* **Yayın (Deployment):** Vercel (Frontend) & Render (Backend)
